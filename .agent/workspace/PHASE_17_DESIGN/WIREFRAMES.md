## Interaction States Summary

### Focus States

All interactive elements must have visible focus states:
- `outline: none`
- `ring: 2px solid accentPrimary`
- `ring-offset: 2px`
- `ring-offset-color: bgDark`

### Loading States

Every async action shows loading feedback:
- Buttons: Spinner icon + "Loading..." text
- Cards: Skeleton placeholders
- Tables: Row skeleton animation
- Full page: Centered spinner with message

### Error States

Consistent error display:
- Form inputs: Red border + error message below
- Cards: Red alert banner at top
- Toast: Red toast with error message
- Full page: Error illustration + retry button

### Empty States

Every list/grid has empty state:
- Illustration (optional)
- Title: "No [items] yet"
- Description: Context-specific help text
- Action button (if applicable)
